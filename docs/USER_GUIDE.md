# User Guide - NEWYM CRM

## Getting Started

### First Steps

1. **Create Account**: Sign up as a trainer
2. **Connect Google Calendar**: Link your Google Calendar for automatic client sync
3. **Add Clients**: Import clients or add them manually
4. **Set Up Pipeline**: Configure your CRM pipeline stages

## Features

### Client Management

#### Adding Clients

1. Navigate to **CRM > Clients**
2. Click **Add Client** button
3. Fill in client information:
   - Name (required)
   - Email
   - Phone
   - Initial status
4. Click **Save**

#### Importing Clients

1. Go to **CRM > Import**
2. Select file format (CSV or JSON)
3. Upload your file
4. Preview the data
5. Click **Import**

#### Client Details

View comprehensive client information:
- Contact details
- Interaction history
- Payment tracking
- Documents
- Calendar events

### Pipeline Management

#### Pipeline Stages

Default stages:
- **Lead** - New potential clients
- **Qualified** - Verified leads
- **Active** - Current clients
- **Inactive** - Paused clients
- **Churned** - Lost clients
- **On Hold** - Temporarily paused

#### Moving Clients

1. Drag and drop clients between stages
2. Or use the status dropdown in client details
3. Add a reason for status change

### Reports & Analytics

#### Pipeline Statistics

View:
- Total clients
- Clients per stage
- Conversion rates
- Pipeline velocity

#### Revenue Reports

Track:
- Total revenue
- Monthly revenue
- Average transaction value
- Overdue payments

#### Activity Reports

Monitor:
- Client interactions
- Follow-up tasks
- Communication history

### Advanced Features

#### Filtering & Segmentation

1. Go to **CRM > Advanced Filters**
2. Add filter conditions:
   - Field (status, payment, etc.)
   - Operator (equals, contains, etc.)
   - Value
3. Apply filters
4. Save as segment for reuse

#### Bulk Actions

1. Select multiple clients
2. Choose action:
   - Update status
   - Update payment status
   - Add tags
   - Export
   - Send email
3. Apply to selected clients

#### Email Templates

1. Go to **CRM > Email Templates**
2. Create new template:
   - Name and description
   - Subject (for emails)
   - Body with variables
3. Use variables: `{{name}}`, `{{email}}`, etc.
4. Preview before sending
5. Send to clients

#### Scheduled Exports

1. Go to **CRM > Scheduled Exports**
2. Create new export:
   - Name and description
   - Format (CSV, JSON, Excel)
   - Data type (clients, interactions, etc.)
   - Schedule (daily, weekly, monthly)
   - Delivery method (email, storage)
3. Enable/disable as needed

### Automation

#### Automation Rules

1. Go to **CRM > Automation**
2. Create rule:
   - Trigger conditions
   - Actions to execute
3. Enable rule
4. Monitor execution

#### Follow-up Reminders

Automatically create tasks for:
- Clients needing follow-up
- Payment reminders
- Appointment reminders

## Keyboard Shortcuts

- `Ctrl/Cmd + K` or `/` - Search
- `Ctrl/Cmd + N` - New client
- `J/K` - Navigate between clients
- `Enter` - Open selected client
- `Esc` - Close modal/dialog

## FAQ

### How do I sync Google Calendar?

1. Go to **Settings > Integrations**
2. Click **Connect Google Calendar**
3. Authorize access
4. Configure sync settings

### How do I export client data?

1. Go to **CRM > Export**
2. Select format (CSV, JSON, Excel)
3. Choose data type
4. Apply filters (optional)
5. Click **Export**

### How do I set up scheduled exports?

1. Go to **CRM > Scheduled Exports**
2. Click **New Scheduled Export**
3. Configure schedule and delivery
4. Save and enable

### How do I filter clients?

1. Use the search bar for quick search
2. Use **Advanced Filters** for complex queries
3. Save filters as presets for reuse

### How do I send bulk emails?

1. Select multiple clients
2. Click **Bulk Actions**
3. Choose **Send Email**
4. Select template
5. Send
